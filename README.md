This is my blog [vitonsky.net](https://vitonsky.net).

Found typo? Make [issue](https://github.com/vitonsky/blog/issues/new/choose) then.

## Setup

Create file `.env` and fill variables, to make scripts work with external services.

## Commands

- `post:add` adds a new blog post
- `post:proofread` fix typos, grammar and text style for new posts in branch
- `post:review` review new posts in branch with AI and get advices
